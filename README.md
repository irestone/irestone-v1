# IRESTONE

My portfolio website.

## Stack

- platform: *Node.js*
- view engine: *Pug*
- scripts: *JavaScript* (bundling with *rollup.js*)
- styles: *Stylus*

## Libs

- [Highway](https://highway.js.org/)
- [GSAP](https://greensock.com/gsap/)

## Usage

`npm start` to run the server or `npm run dev` to run it in development mode.
